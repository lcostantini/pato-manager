# My Todo

a console application to keep track of your todos

## Install
    git clone git@github.com:olvap/pato-manager.git
    cd pato-manager
    bundle

## Usage
    create a new task
    ruby todo.rb new [task_name]

    list all task
    ruby todo.rb all
